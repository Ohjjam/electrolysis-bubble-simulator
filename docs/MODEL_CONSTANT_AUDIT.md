# 모델 상수 감사 — Bubble simulator

기준: 2026-07-20 현재 코드. 숫자를 모두 없애는 것이 목표는 아니다. 아래처럼 출처와 역할을 구분한다.

- **물리 상수**: Faraday 상수, 기체상수, 표준중력처럼 그대로 사용 가능
- **측정 입력**: 접촉각, 막 저항, 교환전류밀도처럼 대상 셀에서 측정해야 하는 값
- **문헌 상관식**: 적용 범위와 출처를 명시하고 범위 밖에서는 경고해야 하는 값
- **수치 설정**: 격자, 시간간격, 입자 수처럼 수렴성 시험으로 정해야 하는 값
- **임의 모델 상수**: 데이터나 유도식 없이 결과를 맞추는 값. 제거·비활성화·측정 입력 전환 대상
- **표시 전용**: 계산에는 영향이 없고 화면 모양에만 쓰이는 값

## 이번 수정에서 제거한 임의 상수

| 기존 값 | 기존 역할 | 변경 |
|---|---|---|
| `drag_K = 60 s^-1` | 기포가 있는 셀의 유속을 임의 감쇠 | 제거. 국소 기체분율, Sauter 평균직경, 기포-액체 상대속도와 Schiller–Naumann 항력계수로 상간 운동량 교환률을 계산 |
| 자유기포 `wobble = 0.35` 및 무작위 crawl | 궤적을 흔들어 보이게 함 | 계산 궤적에서 제거. 해석된 액체속도, 부력 slip, Tomiyama lift, Antal wall lubrication만 사용 |
| 합체확률 `0.05 / 0.9` | 농도에 따라 매 충돌에 임의 확률 적용 | 제거. 기하학적 충돌 커널과 Prince–Blanch 액막 배수 효율 사용. 전해질의 임계합체농도는 별도 실험 입력으로 남김 |
| 접촉거리 `1.6 r sin(theta)` | 벽면 합체 도달거리 | 동일 크기 두 footprint의 합인 `2 r sin(theta)`로 수정 |
| 스텝마다 새 합체 입자 생성 | 시간간격이 작을수록 계산 입자와 시각 이벤트가 증가 | Shima 계열 가중 입자쌍 갱신으로 교체. 실제 기포 수와 기체 체적을 보존하면서 기존 두 cohort를 갱신 |

## 우선순위 0 — 정량 예측 전에 반드시 처리

| 위치 / 항목 | 분류 | 문제 | 권장 조치 |
|---|---|---|---|
| 기존 `fritz_scale=0.08` | **제거 완료** | Fritz 식 결과를 8%로 축소해 전해기포 크기 범위에 맞추던 값이었다. | 3D 기본 경로에서 제거하고 `무유동 이탈직경` 실측 입력으로 교체. 기본 244 µm는 예시값임을 UI에 명시 |
| `bubblesim/config.py: site_density, r_nuc` | **측정 필요 모델 입력** | 3D의 실제 기포 수, 성장시간, 면적 피복률을 직접 결정한다. 현재 기본값은 특정 전극 측정값이 아니다. | 전극별 핵생성 사이트 밀도와 초기 핵 크기 입력/데이터셋화. 값이 없으면 절대 기포 수 대신 정규화 결과로 표시 |
| `bubblesim3d/params3d.py: j0_anode, j0_cathode, r_mem` | **셀 보정값** | 현재 AHEAD 계열 셀 곡선에 맞춘 겉보기 값이며 보편 상수가 아니다. | 프리셋 이름·출처·온도·전해질 범위를 함께 저장. 사용자 셀에서는 EIS/분극곡선 실측값 필수 |
| `void_frac=0.82` | **임의 보정** | 채널 기포가 전해질 저항 경로의 몇 %를 막는지 하나의 값으로 환산한다. 실험 sweep 전압에 직접 영향 | 기본 sweep에서 꺼두거나, 3D 기체분율과 실제 전류 경로로 계산. 당장은 실측 보정값임을 명시하고 셀 프리셋에 종속 |
| `properties.py`의 KOH 외 밀도·점도·표면장력 및 일부 전도도 fit | **거친 물성식** | 선형/간이식이며 H2SO4·인산에서 정량 정확도를 보장하지 않는다. | 온도·농도 범위가 명확한 원 데이터 상관식으로 교체하고 범위 밖 입력 차단 |

## 우선순위 1 — 현상은 표현하지만 실험 또는 더 나은 폐쇄식 필요

| 위치 / 항목 | 분류 | 현재 의미와 한계 | 권장 조치 |
|---|---|---|---|
| `properties.py: c_coalesce` + `parcels.py: p_merge()` | **실험 임계값 + 연속 폐쇄식** | `η=c_crit/(c_crit+c)`로 바꿔 1 M KOH도 합체가 0이 되지 않는다. 새 피팅상수는 없지만 함수형 자체는 이온별 액막 배수를 직접 푼 것이 아닌 1-파라미터 폐쇄식이다. | 전해질·기체·온도별 합체효율 데이터가 생기면 현재 연속식을 데이터 기반 함수와 불확실성 범위로 교체 |
| `parcels.py: mesh Young–Dupré release` | **측정 정적 접촉각 + 무이력 근사** | PP 기포각 101.2°로 정적 부착력을 계산한다. 임의 이탈계수는 없지만 전진·후진각이 없어 접촉선 pinning/rolling resistance는 0인 하한 가정이다. | 실제 mesh에서 전진·후진 기포각 또는 유동별 이탈직경 영상을 측정해 검증 |
| `parcels.py: mesh pitch` | **카탈로그 기하 + 대칭 가정** | 두 개구와 개방률만으로 warp/weft pitch를 각각 유일하게 정할 수 없어 `hole/pitch=sqrt(open)`로 두 방향 차단비를 같게 둔다. 피팅 길이는 없지만 직조 상세가 생략된다. | 제조사 strand 수·pitch 또는 현미경 치수를 입력하면 대칭 가정을 제거 |
| `mesh_mode=hydrophobic` (Mesh 2) | **의도적 격리 가정** | 카탈로그 개구·개방률·두께는 기포 접촉면 판정에 남지만 mesh를 CFD solid voxel로 만들지 않고 채널식의 `obstruction=0`, `u_boost=1`, `dp_ratio=1`로 둔다. 3D의 청록색 평면망도 계산 격자가 아니라 이 무수력 접촉면 표시다. | 실제 mesh의 압력손실을 배제하려는 대조군 용도로만 사용; 실제 셀 압력강하 예측에는 Mesh 1 또는 투과도 데이터 필요 |
| `parcels.py: H0=1e-4, HF=1e-8 m` | **문헌 액막 폐쇄값** | Prince–Blanch 액막 배수식의 초기/파열 막 두께. 보편 물리상수가 아니다. | 계면활성제·전해질별 문헌값 또는 실측값으로 분리하고 현재 적용 범위를 UI에 표시 |
| `Cd_flow=1.2` | **경험 항력계수** | 벽 부착 기포 이탈 힘에 사용. 기포 형상·벽·Re 의존성을 상수 하나로 근사 | 부착 기포 형상/투영면적과 Re 의존 항력 상관식으로 교체 |
| `k_mhd=2e-5` | **임의 경험계수** | `jB`를 속도로 바로 환산한다. 전도도·전극 간격·Lorentz force 해석이 생략됨 | 기본값 0 유지. MHD 사용 시 `J×B` 체적력과 전도도 기반 유동 계산으로 교체 |
| `dep_gradient_length=100 µm` | **임의 DEP proxy** | 균일 전기장 입력만으로는 DEP를 계산할 수 없어 구배 길이를 가정 | 전위장/전극 형상에서 `grad(E^2)`를 계산하기 전에는 DEP 정량 기능 비활성화 |
| `r_min_detach=8 µm`, `detach_spread=0.3` | **모델 바닥값/분산** | 지나치게 작은 이탈반경 방지와 표면 불균일성 표현. 데이터 없이 결과 분포를 바꿈 | 측정 분포 입력으로 전환. 바닥값이 작동하면 진단 경고 출력 |
| `sh0=25`, `k_vogt=0.6`, `flow_jlim=1.5` | **물질전달 폐쇄/보정** | 자연대류·기포 교반·유량 효과가 서로 다른 경험식에 중복될 수 있음 | 3D 농도수송을 풀기 전에는 각 효과를 독립 토글하고 사용 데이터셋에 맞춰 보정 |
| `n_drag=2.5`, `D_w_mem=1e-9`, `c_w_mem=4e4` | **막 재료 입력** | AEM마다 크게 다르며 상수가 아님 | 막 프리셋별 측정값으로 관리; 재료 미선택 시 건식 음극 모델 비활성화 |
| `thermal_mass=50`, `hA_cool=0.5` | **장치 입력** | 셀 크기와 냉각 조건에 종속 | 형상·재료 질량·냉각 유량에서 계산하거나 실측 입력화 |

## 우선순위 2 — 수치 설정과 계산 해상도

이 값들은 임의 물리계수가 아니라 계산 해상도다. 다만 결과가 이 값에 수렴하는지 시험하지 않으면 사실상 모델 상수처럼 작동한다.

| 위치 / 항목 | 역할 | 필요한 검증 |
|---|---|---|
| `server3d_app.py: DT=3 ms` | 시간 해상도 | 1, 2, 3, 4 ms에서 기체보존·합체율·출구직경 비교 |
| `params3d.py: h=2 mm`, `max_cells=60000` | 유동/기체분율 격자 | 최소 3개 격자에서 압력강하·유량·국소 void·d32 수렴성 |
| `parcels.py: N_SITES=140`, `FREE_TARGET=1400`, `cap=6000` | 가중 입자 표본 수 | 700/1400/2800 이상 ensemble 통계와 신뢰구간. 절대 기포 수와 구분 |
| `SUB_FRAC=0.25`, `MAX_SUB=32` | 벽 기포 성장 substep | 이탈 크기·주기 변화가 허용오차 이내인지 확인 |
| `faceredist.py: slab_cells=2, cap=0.9`, footprint 최소 0.51 bin | 전류 재분배 공간 필터 | 격자 독립성 확인. 현재는 국소 전류맵에 영향을 주는 수치 폐쇄 |
| `cell3d.py: PROJ_TOL=2e-3`, `PROJ_ITERS=80` | 압력 투영 종료 조건 | 무차원 발산, 입출구 질량수지 기준을 계속 회귀 시험 |
| `CONF_FRAC=0.45` | 구형 기포가 채널의 90%를 넘지 않게 제한 | 이는 slug-flow 전이 모델이 아니다. 제한에 도달하면 '구형 기포 모델 범위 초과'로 보고하고 더 성장시키지 않음 |

`alpha_g > 1`인 격자는 임의 상한으로 정상처럼 숨기지 않는다. 계산은 발산 방지를 위해 1 직전에서 제한하지만, 진단의 `alpha_g_raw_max`, `alpha_g_overfilled_cells`, `dispersed_bubble_valid=false`와 UI 경고로 분산기포 모델의 적용 범위 초과를 명시한다.

## 표시 전용 상수

`web3d/app3d.html`의 부착 기포 납작함(`normalScale`, `tangentScale`, 중심 이동 0.86), 투명도, 매칭 거리, fade 시간, 합체 애니메이션 시간은 **물리 계산값을 바꾸지 않는다**. 다만 사용자가 실제 계산으로 오해할 수 있으므로 다음 원칙을 적용한다.

1. 자유 기포 변형은 Eötvös 수 기반 Wellek 상관식으로 표시한다.
2. 합체 애니메이션은 서버가 보낸 실제 합체 이벤트에만 생성한다.
3. 부착 기포 형상은 현재 표시용임을 명시하고, 추후 Young–Laplace 형상 해석으로 교체한다.
