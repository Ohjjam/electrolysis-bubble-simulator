# 3D 시뮬레이터 방정식·수치기법 정리 (`bubblesim3d`)

> 2026-07-10 코드 기준. 최신 3세대(3D 셀 시뮬레이터)가 **실제로 풀고 있는 식**과
> **수치 방법**을 모듈별로 정리한 것. 아키텍처 개요는 [`3D_SIMULATOR.md`](3D_SIMULATOR.md),
> 커널 물리의 배경 설명은 [`../README.md`](../README.md) 참조.

## 0. 설계 원칙 — 전기화학은 재구현하지 않는다

3D 엔진은 **전기화학을 다시 유도하지 않는다.** 검증·캘리브레이션이 끝난
`bubblesim.kernel`(순수 Python, 골든 테스트로 동결)을 그대로 import해서 쓰고,
3D에서 새로 짠 것은 **공간(유동·기하·다공구조) 쪽**뿐이다. `bubblesim/`은 무수정.

| 트랙 | 스케일 | 새로 푸는 것 (3D 신규) | 커널에서 가져오는 것 |
|---|---|---|---|
| **A — 셀 (라이브)** | mm, 유로/채널 | 비압축 Navier–Stokes(MAC), Lagrangian 기포 수명주기, 면 전류 재분배, 유로 복셀화 | 2전극 Butler–Volmer 전압수지, Faraday 기체율, 이탈 힘균형, 종단속도, 물성 전부 |
| **B — 포어 (배치→재생)** | µm, 다공 전극 내부 | GRF 미세구조 생성, screened-Poisson 전류분포, 복셀 채움 기포 성장 | 침투깊이(Newman), BV 기울기, Faraday 기체율, 형태학(morphology) 라이브러리 |

## 1. Track A — 셀 스케일 (`cell3d.py` 오케스트레이션)

매 스텝의 커플링 루프:

```
버블 → θ(coverage), ε(근벽 void) ─→ [커널] 2전극 전압수지 → j (8스텝마다 갱신)
버블 → 3×3×3 void 장 ────────────→ [NS] 부력 + Brinkman 차단 → 유동장
유동장 + j ─────────────────────→ [파슬] 핵생성·성장·합체·이탈·상승·배출
```

### 1.1 스칼라 작동점 — 2전극 Butler–Volmer 전압수지
[커널 재사용: `solvers/zerod.py` `ZeroDTwoElectrodeSolver` + `kernel/kinetics.py`]

전지 전압 수지 (CA: V 고정→j 풀이 / CP: j 고정→V 계산):

```
V_cell = E_rev(T,P) + η_act,anode(j) + η_act,cathode(j) + η_conc(j) + j·R_total
```

- **반응속도 (전극별 완전 Butler–Volmer):**
  ```
  j = j0 · [ exp(α_a·F·η/RT) − exp(−α_c·F·η/RT) ]
  ```
  η(j)는 이분법으로 역산(`_invert_bv`, [0, 3 V] 브래킷).
- **버블 커버리지 → 활성면적 차단:** 국소 전류밀도가 `j/(1−θ)`로 올라가
  과전압 페널티 `(RT/α_a F)·ln(1/(1−θ))`가 전극별로 추가됨.
  **θ는 3D 파슬의 실제 접촉 발자국에서 계산**되어 스텁(`_Stub`)으로 주입
  — 기포가 덮으면 전압↑(CP)/전류↓(CA)로 되먹임이 닫혀 있음.
- **농도 과전압 (경계층 물질전달 한계):**
  ```
  η_conc = −(RT/zF)·ln(1 − j/j_lim),   j_lim = Sherwood(Re,Sc) 경계층 + Vogt 증강
  ```
- **저항 직렬합:**
  ```
  R_total = gap/κ_eff + r_membrane_area + r_contact_area
  κ_eff = κ·(1−ε)^1.5   (Bruggeman; ε = 파슬의 근벽 void)
  ```
- **캘리브레이션 기준:** AHEAD 측정 I–V (Zhang, Sci. Adv. 2026; 30 wt% KOH, 80 °C,
  zero-gap) — 기본값 `j0_HER=130`, `j0_OER=1.3e-7`(겉보기), `r_mem=3.2e-6 Ω·m²`.
- 전기화학은 느리게 변하므로 **8 유동스텝마다** 재계산 (CachedSolver 패턴).

### 1.2 전해액 유동 — 비압축 Navier–Stokes
[3D 신규: `ns3d.py`]

```
∂u/∂t + (u·∇)u = −∇p + f_부력 + f_차단      ,   ∇·u = 0
```

| 요소 | 방법/식 |
|---|---|
| 격자 | **Staggered MAC** (u,v,w는 면, p는 중심) — 발산·압력구배 연산자가 일치해 투영 후 ∇·u ≈ 기계정밀도 (collocated 격자의 checkerboard 모드 회피) |
| 시간 전진 | **Chorin projection**: 힘 추가 → 투영 → semi-Lagrangian 이류 → 투영 |
| 압력 Poisson | `∇²p = ∇·u*·h²` 를 **red-black SOR**(ω=1.7)로, 이전 스텝 p로 **warm-start** |
| 이류 | **Semi-Lagrangian** (back-trace + trilinear 보간) — 무조건 안정 |
| 부력 (기포→액체) | Boussinesq형: `Δu = g·buoy·α_gas·ĝ(tilt)·Δt` (buoy=0.6, α_gas = 파슬이 뿌린 void). **중력 벡터 ĝ가 tilt 각으로 회전** → 셀 기울임 효과가 물리에서 창발 |
| 차단 항력 (기포→액체) | **Brinkman형 벌점**: `u ← u/(1 + K·α_gas·Δt)`, K=60 s⁻¹ — 기포 커튼 속 액체가 운동량을 잃어 흐름이 기포 구름을 **우회** (grid-scale Euler–Euler 항력) |
| 감쇠 | 선형 damp 3 s⁻¹ + semi-Lagrangian 수치확산 (**명시적 점성항 없음** — Stam stable fluids; 정성 스코프) |
| 경계 | 전극면·측벽·리브: 법선속도 0 (장애물 마스크는 압력 스텐실에서 이웃 제거 = 내부 Neumann 벽), 하단 **인렛 포트**에 u_in, 상단 p=0 개방 outlet |

### 1.3 기포 — Lagrangian 파슬 + 대표 다중도
[3D 신규 골격 + 커널 물리: `parcels.py`]

커널 `Surface`의 수명주기(핵생성→성장→이탈→상승)를 3D로 리프트. 추적 버블 1개가
실제 버블 `mult`개를 대표: **r은 진짜 단일 버블 반경**(문헌 스케일 수십~수백 µm),
가스량은 `W = mult·V(r)`. 실제 셀의 초당 10⁵~10⁶개 이탈(추적 불가능 벽)을 정직하게 흡수.

| 단계 | 식/방법 |
|---|---|
| **핵생성** | Faraday 기체 예산으로 시드 구매: `Q̇ = η_F·(j·A)/(zF)·RT/P` (HER z=2, OER z=4 → H₂가 같은 전하당 2배 부피; anode gas_factor 0.5). 시드 r=2 µm, **촉매면**(zero-gap이라 채널의 막쪽 벽)의 연속 랜덤 위치, 랜드 아래는 기각샘플링으로 배제. `mult = site_density·(0.5+β/90)·A / 140슬롯` |
| **성장** | 남은 예산을 부착 버블에 **r² 가중 분배**: `dV_single = dW/mult` → `r_new = (3V/4π)^{1/3}`. 성장-이탈 주기(~1 ms)가 유동 스텝(3 ms)보다 짧으므로 **서브스텝**(주기 가스량의 25%/패스, ≤32회). r이 이탈반경·채널구속에 걸리면 잉여 가스는 `mult` 재유도(= 같은 크기의 실제 버블이 더 많아짐)로 흡수 — W 보존 |
| **합체** | 통계적: 이웃(자기 mult 형제들, Poisson 분포 n = mult·N/A)이 접촉 반경 `d = 1.6·r·sinβ` 안에 들어올 확률 `P = 1−exp(−π n d²)`의 **성장 증분 dP**에 대해 추첨(dt 무관 = rate-consistent). 합체 확률은 전해질 규칙: KOH > 0.3 M이면 salting-out으로 0.05, 아니면 0.9. 합체 시 `mult → mult/2`, `r → r·2^{1/3}` (W 불변) |
| **이탈** | 커널 힘균형 `F_b + F_d + F_DEP = F_adh`의 해 r_d (이분법): 부력 `(4/3)πΔρg·r³`, 유동항력 `½C_d ρ_l u_eff² π r²`, 음의 DEP `∝ E²·r³`(부피 스케일링), 접촉선 부착력 `A_adh·r` — A_adh는 무유동·무장에서 정확히 **Fritz 반경**에 이탈하도록 보정: `r_d0 = fritz_scale·½·0.0208·β[deg]·√(σ/(gΔρ))` (fritz_scale=0.08: Fritz는 비등 상관식이라 전해에 과대). MHD는 `u_eff = u + k_mhd·j·B`. **벽전단 보정**: 정지액에서 r 먼저 풀고 `u_eff = u_bulk·min(1, r_stag/(gap/2))`로 재평가(이탈 버블은 벽 전단층 안에 있으므로). 버블별 ±35% 분산, 채널 구속 상한 `r ≤ 0.45·gap`(그 이상은 슬러그) |
| **상승** | 풀린 유동 속도(trilinear 샘플) + **Schiller–Naumann 종단 슬립**: `C_d = (24/Re)(1+0.15·Re^0.687)`, `U = √(8Δρ g r/(3 C_d ρ_l))` 감쇠 고정점 반복(Stokes 시드) + 약한 횡방향 미앤더. **복셀 충돌은 축별·서브스텝**(≤반셀/이동)이라 어떤 속도에서도 리브 관통 불가 → 버블이 리브 밑을 미끄러져 턴 갭으로. 상승이 막힌 버블은 느린 횡 crawl(접촉선 wobble). 출구 도달 시 배출(vent) |
| **되먹임** | 가스 부피를 **3×3×3 스미어**로 격자 void에 투영(≤0.8) → NS 부력·차단 항력의 α_gas |
| **커버리지** | Poisson 합집합: `θ = 1 − exp(−Σ mult·π(r·sinβ)²/A)` (≤0.95) → 1.1의 전압수지로 |
| **보존** | `produced(∫Q̇dt) = resident(ΣW) + vented` **기계정밀 폐합** — 갈 곳 없는 가스는 pending으로 이월, 자유 버블 솎아내기(1400개 유지)는 생존자의 W·mult 동일 배율 재규격화 |

### 1.4 전극면 전류 재분배 — 병목맵
[3D 신규, face2d 방식: `faceredist.py`]

근벽 버블들의 실제 접촉 발자국 `mult·π(r sinβ)²`을 유동격자 3×3배 해상도 빈에
**디스크로 스탬핑** → Poisson 합집합 `θ = 0.9(1−exp(−면적/빈면적))` → 3×3 블러 2회 →

```
j(y,z) = j_mean · (1−θ(y,z)) / mean(1−θ)        (활성 셀 평균 = j_mean, 전하 보존)
```

리브(랜드) 셀은 j=−1로 표시(전해액 없음). 정직한 한계: 공통 과전압 가정의
전극-네트워크 축약이지, 3D 갭 퍼텐셜 풀이가 아님.

### 1.5 유로 기하 복셀화
[3D 신규: `geometry.py`, `params3d.py`]

- 디자이너(serp/par/inter, 채널폭·랜드폭·채널수·깊이) → 격자 장애물 마스크.
  서펜타인은 **균일 두께 랜드 밴드 + 끝단 교대 턴 갭** = 연결된 한 줄 뱀길;
  interdigitated는 홀짝 채널 막단(dead-end); 채널은 최소 1열 개방 보장.
- **Zero-gap MEA**: 채널 층(designer 채널 깊이 → n_lay 복셀) 사이의 코어(PTL+막)는
  전부 solid — 액체는 채널에만. 촉매면 = 코어 경계(채널의 막쪽 벽).
- **인렛 포트**: 서펜타인 첫 패스의 시작단만 급수(전체 바닥면 급수 시 턴 갭에서
  면적비 ~10×의 제트가 생겨 버블이 리브를 넘는 오류 — 수정됨).
- 라이브 뷰 = 계산 격자 그대로(과장 배율 없음), 렌더러 리브는 물리 랜드 마스크에서 생성.

## 2. Track B — 포어 스케일 (`pore3d.py` 오케스트레이션)

프레임마다: 전류분포 재풀이(가스가 포어 막음 반영, 4프레임마다) → 핵생성 가중치 →
Faraday 가스량만큼 복셀 채움 성장 → 탈출면 도달 클러스터 배출.

### 2.1 미세구조 생성 — leveled-wave Gaussian random field
[3D 신규: `microstructure.py` + 커널 `morphology` 라이브러리 목표치]

```
백색잡음 → FFT 저역필터 exp(−2(π·corr)²k²) → 정규화 필드 f
solid = |f| < δ,  δ = quantile(|f|, solid_frac)     ← 목표 공극률 정확히 일치
```

- 레벨셋 주변 밴드 = **이중연속**(bicontinuous) 스트럿/포어 네트워크 — 금속상(전자)과
  포어상(이온·가스)이 **둘 다 관통**해야 전류·가스 솔버가 성립. flood-fill로 검증.
- 연결성 하한 `solid_frac ≥ max(0.03, 9/n)` (64³에서 폼 95% 공극률은 스트럿이
  복셀보다 얇아짐 → 하한 적용, `eps_achieved`로 명시; 128³에서 회복).
- 상관길이: carbon_paper n/14 < ni_mesh n/10 < foam n/8 (기재별 스트럿 굵기).
- 비표면적은 solid|pore 계면 면 수로 측정해 라이브러리 `a`와 대조.

### 2.2 전류분포 — 선형화 다공전극 (screened Poisson)
[3D 신규 솔버 + 커널 침투깊이: `current3d.py`]

열린 포어상의 전해질 퍼텐셜 ψ:

```
∇²ψ = ψ / λ²          (modified Helmholtz; 금속상은 고 σ_eff 등전위 근사)
ψ = 1 (분리막/접근면 Dirichlet), 무플럭스 벽
```

- **침투깊이** (Newman 다공전극 고전 결과, 커널 `morphology.penetration_depth`):
  ```
  λ = √( σ_eff·κ_eff / ((σ_eff+κ_eff) · a · dj/dη) )
  ```
  `dj/dη`는 η=0.1 V에서 커널 Butler–Volmer(×R_f)의 수치 기울기.
  λ ≫ 두께면 전극 전체 활용, λ ≪ 두께면 표면 스킨만 반응.
  (표시 가독성 위해 [n/8, 4n] 복셀로 클램프 — 문서화된 모델링 선택)
- 풀이: **RB-SOR** — `ψ = Σ_이웃ψ / (n_이웃 + 1/λ²)`, [0,1] 클립.
- **표면 전류**: 반응 표면 복셀(포어∩솔리드 접촉)에서 `I_voxel ∝ ψ`,
  총합 = 부과 전류(정전류)로 정규화 — **전하 보존**.
- **가스→전류 되먹임**: 가스가 찬 포어 복셀은 전도·반응에서 제외 → 전류가
  아직 열린 채널로 재분배.

### 2.3 기포 성장 — 복셀 채움 (VOF 아님)
[3D 신규: `poregrowth.py` + `gastransport3d.py`]

```
목표 가스 복셀 수 = round( (produced − vented) / V_voxel ),  produced += Q̇_Faraday·dt
```

1. **성장 우선**: 기존 가스 클러스터를 열린 포어로 6-연결 팽창(고가중 프런티어 우선)
   — 물리 그림: 버블이 포어 벽을 만나면 그 연결 채널을 따라 차오름.
2. **핵생성**: 남으면 가중치 상위 표면 복셀에 시드.
   가중치 `w ∝ max(0, I_surface_voxel) · nuc_site_mult` (전류 많은 곳 × 기재 site 배수).
3. **배출**: 탈출면(분리막쪽, 전해액이 향하는 면)에 6-연결로 닿은 가스 클러스터가
   벤트 — 폼은 `escape_factor < 1`로 일부 갇힘.
4. **보존**: `produced = resident + vented` (복셀 1개 이내 오차).

Henry 용존 포화 `c_sat = P·k_H`, 과포화 `S = c/c_sat`도 커널 transport에서 제공
(핵생성 편향의 물리 근거).

## 3. 커널에서 공급되는 물성·상관식 (3D가 쓰는 것)

`build_context(op, params)`가 온도·농도·전해질(KOH/H₂SO₄/PB)에 맞춰 번들:

| 항목 | 내용 |
|---|---|
| `E_rev(T,P)` | 가역전압 1.229 V 기준, dE/dT ≈ −0.9 mV/K, 압력 Nernst 항 (옵션 `high_fidelity`: 수증기압·물 활동도 보정) |
| `κ(c,T)` | 전해질 전도도 (KOH ~6 M 최대; `high_fidelity`: Gilliam 2007 상관식) |
| `σ, μ, ρ_l, Δρ` | 표면장력·점도(농축 KOH 초선형)·밀도차 → Fritz·부력·종단속도에 |
| `j0(T,c), α_a, α_c` | 전극별 교환전류밀도(Arrhenius 열활성) — AHEAD 캘리브레이션 기본값 |
| `j_lim` | 경계층 물질전달 한계: `Sh = f(Re, Sc)` → 유속 의존 + Vogt 기포 증강 |
| `r_membrane_area, r_contact_area` | 막·접촉 직렬 저항 |
| `fritz_scale, k_mhd, Cd_flow, r_min_detach` | 이탈 힘균형 계수 |
| `site_density` | 핵생성 site 밀도 (× 젖음성 인자 0.5+β/90) → `mult` 유도 |
| `morphology` | 기재×나노구조 → R_f, a, ε_p, σ_eff, L_e, escape_factor, nuc_site_mult (Track B) |

## 5. 정직한 한계 (모듈 docstring에 동일 명시)

### Track A — 셀 스케일
- **유동**: 명시적 점성항 없음(Stam 방식: 수치확산 + damping) → 부력 플룸·자가교반·
  기울임 가스갇힘의 **정성** 시각화. 경계층 해상 CFD 아니며, 채널 압력강하·Re는 정량값이 아님.
- **기포**: sub-grid Lagrangian 파슬 — 계면 추적(VOF) 아님.
  - **이탈 후 성장 없음** (성장은 벽에 붙어 있는 동안만).
  - **벽면-법선 이동 없음**: 부력은 전극면과 나란하고 전단 유도 양력(Tomiyama)·난류 분산
    모델이 없어, 이탈한 기포는 벽을 따라 미끄러진다(측정: 셀을 23 mm 거슬러 오르는 동안
    벽면 거리 변화 ±20 µm). ~40 µm 기포가 층류 병류에서 벽에 쌓이는 **방향은 실제와 같지만**,
    기포 커튼이 하류로 두꺼워지는 과정은 재현하지 못한다.
  - **합체는 벽면만**: 접촉 확률 증분식(dt 무관)으로 벽 기포 합체를 풀지만,
    **자유 상승 무리(swarm) 내부의 합체는 미해상** — 필름 배수 효율(collision efficiency)
    모델이 없어 순진한 충돌 커널을 넣으면 스텝당 ν·dt ≈ 40으로 무한정 조대화한다.
  - **성장 부스텝 상한 32**: 씨앗→이탈 사이클(~1 ms)이 유동 스텝(3 ms)보다 짧아 벽 배열만
    부스텝하는데, 아주 얇은 채널·고전류에서는 상한에 걸려 사이클이 미해상이 된다
    (이때 부착 기포 수·피복률 θ가 0에 가깝게 나오는 것은 아티팩트가 아니라 그 정보의 부재).
- **이탈 반경 하한(`r_min_detach`)**: 1 mm 채널·펌프 1 m/s에서 힘균형이 요구하는 크기가
  커널이 모델링하는 최소값보다 작아 **바닥값에 닿는다**(8 µm). 물리적 이탈 크기가 아니라
  **모델 바닥**이다. u ≤ 0.6 m/s에서는 바닥과 무관하게 매끄럽게 변한다.
- **면 전류맵**: 공통 과전압 전극-네트워크 축약(θ 기반 재분배). 3D 갭 퍼텐셜 풀이 아님.
- **포트**: 입·출구는 y=0 / y=Ly **양 끝 면에만** 놓을 수 있다(측면 포트 불가).
  NS의 유입은 지정 법선속도, 유출은 Dirichlet p=0 경계라는 구조에서 온 제약.
- **직접 그린 유로**: 32×32 마스크를 물리 격자에 최근접 리샘플 → 격자가 마스크보다 고우면
  계단이 남는다. 전부 리브로 칠하면 채널 한 줄을 강제로 열어 둔다(유체 셀이 없으면 풀이 불가).
- 스칼라 전기화학은 **0D** (2세대 앱의 다공 깊이/1D/채널 솔버와는 별개 축약 수준).

### Track B — 포어 스케일
- **전류**: 선형화 단상 퍼텐셜(금속 등전위) — 양상 완전 Newton+BV 아님.
  "어디서 기포가 생기나"를 정하는 침투/재분배 물리는 포착.
- **기포 성장**: 복셀 채움(volume-filling) — 계면 추적 아님.
- **미세구조**: 통계적 대표 구조(목표 공극률 정확) — 특정 전극의 토모그래피 재구성 아님.
  폼은 64³ 해상도에서 연결성 유지를 위해 고체분율 하한을 적용(`eps_achieved`로 명시).

## 6. 검증 앵커 (tests/) — 226 테스트

숫자는 모두 현재 코드에서 실측한 값이다.

- **골든 동결**: `bubblesim/` 무수정 → `test_characterization` rel 1e-9로 bit-identical.
- **보존 (정확한 것)**
  - 가스 폐합 `produced = resident + vented` — Track A **~3e-14** (기계정밀),
    Track B 복셀 1개 부피 이내.
  - 대표 다중도 항등식 `W = mult·V(r)` — 성장·구속 clamp·합체·솎아내기 전 구간에서 rel 1e-9.
  - 면 전류 재분배: 활성 셀 평균 = `j_mean` (rel 1e-9).
- **보존 (수렴에 의존하는 것 — 기계정밀 아님)**
  - 압력 풀이: **차가운 닫힌 상자**에서 RB-SOR 1000 sweep → max|div| 3.5e-14 (기계정밀),
    200 sweep → 1e-8.
  - **라이브 루프**(warm-start, 12 sweep, 사행 유로): max|div| ≈ 4.4 s⁻¹, v_max 0.70 m/s
    → 속도 단위 상대오차 **div·h/v_max ≈ 1.3 %**, 유입/유출 질량수지 **0.4 %**.
    테스트는 5 % 이내를 요구한다. 기계정밀은 라이브 루프의 성질이 아니다.
- **물리 범위 (엔진의 실제 설정에서)**
  - 이탈 반경, θ=30/60/120° · **정지 액체**: 61 / 122 / 244 µm (젖음성 레버 4×).
  - 같은 각, **펌프 0.35 m/s · 채널 깊이 1 mm**: 32 / 40 / 43 µm — 큰 기포일수록 전단층에
    더 들어가 항력을 더 받으므로 레버가 **1.3×로 자기제한**된다.
  - 유속 스윕(θ=60°): 122 → 79 → 40 → 15 µm (u = 0 / 0.2 / 0.35 / 0.6 m/s), 단조 감소.
  - 종단속도 = 커널 Schiller–Naumann과 일치(rel 1e-3), 촉매면(막쪽)에서 핵생성,
    부착 기포는 자기 반경만큼 벽에 접함, 미세구조 공극률·관통성.
  - 벽 구속: 어떤 기포도 0.45·채널깊이를 넘지 않음. 합체 횟수는 dt에 무관(4× 세분해도 ±2.5배 이내).
- **외부 검증**: 2전극 기본값이 AHEAD 측정 분극곡선 재현(2.0 V에서 0.785 A/cm²,
  1.6~2.1 V 구간, `test_validation`) — 상세는 [`../VALIDATION.md`](../VALIDATION.md).
