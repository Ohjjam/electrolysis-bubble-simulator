| 전체 테스트 | `py -3.14 -m pytest tests/ -q` (210+ 테스트) |

## 3. 실행 파일 사전 (.bat / .url 전부)

**루트 (3세대 — 지금 쓰는 것):**

| 파일 | 하는 일 |
|---|---|
| `run3d.bat` | 3세대 서버(:8766)를 콘솔 창으로 실행 — 로그 표시, 창 닫으면 종료 |

(숨김 실행용이던 `3D 서버 켜기/끄기.bat`은 7/14 `_구버전 보관/`으로 이동 — run3d.bat만 쓰면 불필요)

**`2세대 라이브앱/` 폴더:**

| 파일 | 하는 일 |
|---|---|
| `서버 켜기.bat` | 2세대 서버(:8765) 숨김 실행 + 브라우저 오픈 (이미 켜져 있으면 링크만 오픈) |
| `서버 끄기.bat` | 2세대 서버 종료 + cloudflared/ngrok 터널도 종료 |
| `외부 링크 켜기.bat` | 외부 공유 터널 시작 (`server_tunnel.py`: ngrok 우선, cloudflared 예비) |
| `전기화학 버블 시뮬레이터.bat` | 종합 메뉴(1+2세대): [1] 라이브 앱 [2] 1세대 HTML [3] 텍스트 데모 [4] 그림 4장 [5] 테스트 [6] 콘솔 [7] 의존성 설치 |
| `publish.bat` | 웹 배포판 빌드(`build_web.py`→`docs/`) 후 **git commit + push** (GitHub Pages 배포용) — 커밋이 포함되니 주의 |
| `버블 시뮬레이터.url` | http://localhost:8765/ 바로가기 (서버 켜져 있을 때만) |
| `버블 시뮬레이터 (외부공유).url` | ngrok 고정 외부주소 바로가기 (터널 켜져 있을 때만) |

모든 .bat은 Python 3.14 전체 경로(`%LOCALAPPDATA%\Python\pythoncore-3.14-64`)를 사용 — `py` 기본이 3.9라서 그런 것이니 바꾸지 말 것.
